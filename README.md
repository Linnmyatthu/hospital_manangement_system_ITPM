## 🚀 How to Set Up and Run

### 1. Clone the repository
```bash
git clone https://github.com/Linnmyatthu/hospital_manangement_system_ITPM.git
```

### 2. Install Flask
```bash
pip install flask
```

### 3. Run the application
```bash
python app.py
```
